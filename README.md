# doc-genie
The system analyzes the structure of a given Python function, extracts important information such as function name, parameters, and return types, and generates well-formatted docstrings in both Google and NumPy styles.

Doc-Genie provides an interactive web interface built using Gradio, allowing users to paste Python functions and instantly receive structured documentation. It also supports exporting the generated documentation as TXT and PDF files for professional use.

The project aims to simplify the documentation process, improve code readability, and maintain consistent documentation standards across software projects.

Technologies Used

Python 3.8+ – Core programming language

AST (Abstract Syntax Tree) – For code structure analysis

Gradio – For building the web interface

ReportLab – For PDF generation

Google Colab – Development and testing environment
